## Câu trả lời

1. Câu 1: REST API thường trả về toàn bộ dữ liệu của tài nguyên nên payload lớn hơn, còn GraphQL chỉ lấy đúng các trường cần thiết nên payload nhỏ hơn, tiết kiệm băng thông và tối ưu hơn cho ứng dụng.

2. Câu 2: Nếu muốn thay đổi giá của một sản phẩm qua Headless API, em sẽ dùng Mutation vì thao tác này làm thay đổi dữ liệu trên server, còn Query chỉ dùng để lấy dữ liệu mà không chỉnh sửa gì.
